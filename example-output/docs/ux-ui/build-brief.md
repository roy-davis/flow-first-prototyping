## Product

Native Android TV audiobook player for SMB and first-class cloud sources. The app indexes remote audiobook libraries into a local cache, then plays media by resolving provider-backed playable asset references on demand.

## Core flow

Home -> Now Playing. Home -> Library -> Book Detail -> Now Playing. Home/Library -> Sources -> SMB Setup or Provider Authorization -> Folder Picker -> Sync Progress -> Discovered Books -> Library. Any source/playback failure -> Source Repair -> origin.
