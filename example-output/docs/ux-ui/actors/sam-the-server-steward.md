# Sam the Server Steward - SMB source owner

> File: docs/ux-ui/actors/sam-the-server-steward.md
> Actor type: Human

**Composite summary:** Sam maintains a home NAS or Windows/Samba share and wants the TV app to connect to a known audiobook folder without becoming a raw file manager. He can provide host, share, username, and password, but expects precise validation, secure credential storage, and clear repair if network or credentials change.
