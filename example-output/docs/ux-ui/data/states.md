## Object: Source

### States

| ID | Name | Description | Entry | Exit |
| --- | --- | --- | --- | --- |
| S1 | no_source | No remote source configured | Fresh install or all sources removed | Add source |
| S2 | draft | Source details being entered | Choose source type | Test/authorize/cancel |
| S3 | testing | SMB validation or provider check in progress | Test connection | Success/failure/cancel |
| S4 | authorizing | Provider authorization pending | Start authorization | Success/error/cancel |
| S5 | connected | Source reachable and authorized | Validation/auth success | Folder selection/sync/repair |
| S6 | syncing | Sync Worker indexing source | Select folder | Complete/partial/failure |
| S7 | partial | Some items skipped or failed | Sync partial result | Retry/review/view library |
| S8 | unavailable | Source cannot be reached | Network/provider/playback failure | Retry/repair |
| S9 | auth_expired | Provider token invalid | Provider auth failure | Reauthorize |
| S10 | permission_blocked | Folder/file access denied | Folder listing/sync failure | Choose another/repair |

### Transitions

| From | Trigger | To | Actor | Guard |
| --- | --- | --- | --- | --- |
| no_source | Add source | draft | Sam/Clara | Always |
| draft | Test SMB connection | testing | Sam | Required fields valid or anonymous selected |
| draft | Start provider authorization | authorizing | Clara | Internet available |
| testing | Validation success | connected | Provider API | SMB reachable and authorized |
| authorizing | Auth success | connected | Provider API | Token/session valid |
| connected | Select folder | syncing | Sam/Clara | Folder readable |
| syncing | Complete | connected | Sync Worker | No blocking errors |
| syncing | Partial errors | partial | Sync Worker | Some usable metadata exists |
| connected/partial | Network/source failure | unavailable | Provider API | Source not reachable |
| connected/partial | Token expired | auth_expired | Provider API | Provider auth invalid |

### Actions per state

| State | Allowed | Blocked | Hidden |
| --- | --- | --- | --- |
| no_source | Add source | Play/browse real books | Playback controls |
| draft | Edit fields, cancel, test/authorize | Indexing | Repair |
| testing | Cancel | Duplicate test | Folder picker |
| authorizing | Cancel, retry after timeout | Folder picker | Credentials fields |
| connected | Choose folder, sync, play | Repair unless degraded | Auth repair |
| syncing | View progress, run in background | Start unsynced items | None |
| partial | View library, retry sync, review source | Hide warning | None |
| unavailable | Retry, repair, view cached library | Start affected playback | Secrets/raw details |
| auth_expired | Reauthorize, view cached library | Folder browsing/playback for source | Raw token |
| permission_blocked | Choose another folder, repair | Sync blocked path | Play blocked files |

### Diagram

```
[no_source] --(Add source)--> [draft]
[draft] --(Test SMB)--> [testing] --(success)--> [connected]
[draft] --(Start auth)--> [authorizing] --(success)--> [connected]
[connected] --(Select folder)--> [syncing] --(complete)--> [connected]
[syncing] --(partial)--> [partial]
[connected/partial] --(network failure)--> [unavailable]
[connected/partial] --(token expired)--> [auth_expired]
```

### Terminal states

| State | Description | UI behaviour |
| --- | --- | --- |
| connected | Source usable | Show normal setup/library/playback paths |
| unavailable | Source failed | Show Source Repair with retry and cached fallback |
| auth_expired | Provider repair required | Show Reauthorize provider |
| permission_blocked | Folder/file blocked | Show choose another/repair |

## Object: Library

### States

| ID | Name | Description | Entry | Exit |
| --- | --- | --- | --- | --- |
| L1 | loading_cache | Reading local metadata | Open Library | Loaded/empty/error |
| L2 | loaded | Cached books available | Cache read success | Open book/start playback |
| L3 | empty | No supported books indexed | Cache read returns zero | Add source/retry sync |
| L4 | stale | Cached books visible but source degraded | Source health warning | Repair/source retry |
| L5 | cache_error | Local metadata cannot be read | Database failure | Retry/report issue |

### Transitions

| From | Trigger | To | Actor | Guard |
| --- | --- | --- | --- | --- |
| loading_cache | Cache read success with books | loaded | App | Local database readable |
| loading_cache | Cache read success with zero books | empty | App | No indexed books |
| loaded | Source degradation | stale | Provider API/Sync Worker | Cached books still readable |
| stale | Repair success | loaded | Sam/Clara/Provider API | Source reachable |
| loaded | Select book | Book playable/unavailable | Leo | Book exists |

### Actions per state

| State | Allowed | Blocked | Hidden |
| --- | --- | --- | --- |
| loading_cache | Wait, Back | Select book | Book grid until data |
| loaded | Browse, open detail, resume focused | N/A | Setup prompts unless secondary |
| empty | Add source, retry sync | Open book | Book grid |
| stale | Browse cached, repair source | Remote-only new content | Raw provider errors |
| cache_error | Retry, exit | Browse | Book grid |

### Diagram

```
[loading_cache] --(books found)--> [loaded]
[loading_cache] --(no books)--> [empty]
[loaded] --(source degraded)--> [stale]
[stale] --(repair success)--> [loaded]
[loading_cache] --(database error)--> [cache_error]
```

### Terminal states

| State | Description | UI behaviour |
| --- | --- | --- |
| loaded | Library usable | Show grid |
| empty | No books | Show Add source |
| cache_error | Cache unavailable | Show retry/error |

## Object: Playback

### States

| ID | Name | Description | Entry | Exit |
| --- | --- | --- | --- | --- |
| P1 | idle | No active playback | App start/no selected book | Start/resume |
| P2 | starting | Resolving playable asset | Resume/start selected | Buffering/error/cancel |
| P3 | buffering | Stream buffer filling | Asset resolved or network refill | Playing/error |
| P4 | playing | Audio active | Buffer ready | Pause/seek/back/error |
| P5 | paused | Audio paused with saved position | Pause selected or session paused | Play/seek/exit |
| P6 | error | Stream failed | Resolution/stream failure | Retry/repair/back |
| P7 | background | Audio continues outside app | Android dashboard exit | Return to player/stop |

### Transitions

| From | Trigger | To | Actor | Guard |
| --- | --- | --- | --- | --- |
| idle/paused | Resume/start | starting | Leo | Playable book selected |
| starting | Asset resolved | buffering | Playback Engine | Source reachable |
| buffering | Buffer ready | playing | Playback Engine | Enough buffer |
| playing | Pause | paused | Leo | Always; save position immediately |
| playing/paused | Dashboard exit | background | System | MediaSession active |
| starting/buffering/playing | Stream failure | error | Playback Engine | Source or codec failure |
| error | Retry playback | starting | Leo | Retry allowed |

### Actions per state

| State | Allowed | Blocked | Hidden |
| --- | --- | --- | --- |
| idle | Start/resume selected book | Pause/seek | Speed menu |
| starting | Cancel/back | Duplicate start | Repair unless error |
| buffering | Pause/stop, wait | Unsafe seek if needed | None |
| playing | Pause, seek, speed, back | Start duplicate | Repair unless degraded/error |
| paused | Play, seek, speed, back | N/A | N/A |
| error | Retry, repair, back | Continue playback | N/A |
| background | Return to player | N/A | In-app detail controls |

### Diagram

```
[idle] --(Resume/Start)--> [starting] --(asset resolved)--> [buffering] --(ready)--> [playing]
[playing] --(Pause)--> [paused] --(Play)--> [playing]
[playing] --(Dashboard)--> [background]
[starting/buffering/playing] --(failure)--> [error] --(Retry)--> [starting]
```

### Terminal states

| State | Description | UI behaviour |
| --- | --- | --- |
| playing | Playback active | Show Now Playing controls |
| paused | Position saved | Show resume controls |
| error | Playback blocked | Show Retry and Repair |
| background | Playback outside app | Return restores player |

## Object: Discovered Book

| ID | Name | Description | Entry | Exit |
| --- | --- | --- | --- | --- |
| D1 | discovered | Indexed from a source but not yet in listening library | Sync finds supported book | Review/add/skip |
| D2 | added | Book exists in listening library | Add to library succeeds | Open in Library |
| D3 | unsupported | Book cannot be added or played | Sync detects unsupported file/book | Repair/skip |
| D4 | duplicate | Book appears to match an existing library item | Sync detects duplicate | Keep existing/review |

### Actions per state

| State | Allowed | Blocked | Hidden |
| --- | --- | --- | --- |
| discovered | Open details, Add to library | Play until added if product requires curation | Raw source path |
| added | Open in Library, Open details | Add duplicate | N/A |
| unsupported | Review issue, choose another source/folder | Add to library | Play |
| duplicate | Open existing, review duplicate | Add duplicate | N/A |
